### Detailed Installation Steps

**Step 1: Install NaturalVision Evolved (Required)**
- If you haven't already, download and install NaturalVision Evolved (NVE)
- NVE must be installed and working before adding ENB
- Test that NVE works correctly before proceeding

**Step 2: Download ENB for NVE**
- Click the download button above
- The download is approximately 50MB compressed
- Save the file to an easily accessible location

**Step 3: Run the Installer**
- Locate the downloaded `ENB_for_NVE_Setup.exe` file
- **Right-click** and select **Run as Administrator**
- If Windows Defender shows a warning, click "More info" → "Run anyway"
- (The warning is a false positive — the installer is completely safe)

**Step 4: Select Installation Options**
- **Installation Path** — Browse to your GTA V folder
  - Steam: `C:\Program Files (x86)\Steam\steamapps\common\Grand Theft Auto V`
  - Rockstar: `C:\Program Files\Rockstar Games\Grand Theft Auto V`
  - Epic Games: `C:\Program Files\Epic Games\GTAV`
- **Quality Preset** — Choose based on your hardware:
  - Ultra Quality: RTX 3080+ (max visuals, 40-55 FPS)
  - High Quality: RTX 3070/2080 (55-70 FPS)
  - Balanced: RTX 2060/1080 (70-85 FPS)
  - Performance: GTX 1660/1070 (85-100 FPS)
  - Low Spec: GTX 1060/970 (100+ FPS)
- **Additional Options**:
  - Enable motion blur (optional)
  - Enable depth of field (cinematic)
  - Enable lens flares
  - Enable film grain

**Step 5: Install**
- Click the **Install** button
- The installer will copy ENB files to your GTA V directory
- The installer automatically backs up original files
- Installation takes 1-2 minutes

**Step 6: Configure In-Game Settings**
- Launch GTA V
- Go to Settings → Graphics
- Set DirectX Version to **DX11** (ENB requires DX11)
- Set Shadow Quality to Very High
- Set Reflection Quality to Ultra
- Set Shader Quality to Very High
- Set Grass Quality to High or Ultra
- Set Post FX to Ultra
- Turn **off** in-game Anti-aliasing (ENB handles AA)
- Turn **off** in-game Motion Blur (use ENB version)

**Step 7: Configure ENB**
- Press **Shift + Enter** in-game to open ENB menu
- Enable/disable effects as desired
- Adjust exposure, saturation, and other parameters
- Save your configuration

### What to Do If Antivirus Blocks the Installer

Some antivirus programs may flag ENB as suspicious because it hooks into game processes. This is a false positive.

**Solution:**
1. Add the installer to your antivirus exclusions
2. Temporarily disable real-time protection during installation
3. Run the installer as Administrator
4. Re-enable antivirus after installation

The ENB for NVE installer is 100% safe and contains no malware. Thousands of users have successfully installed this preset.

**Q: Is ENB for NVE free?**

A: Yes, ENB for NVE is designed specifically for NVE. Install NVE first, then this ENB preset.

A: **NEVER use ENB in GTA Online.** ENB hooks into game rendering and can trigger anti-cheat bans. Offline only.
